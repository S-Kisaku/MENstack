# プロジェクトの開始方法（以下のコマンドを打ってね!）
1.infraフォルダーに入る
```
cd app/infra
```
2.
```
make init
```
---
# アプリの構成
## バックエンド
- DB - MySQL
  * username:dev
  * password:dev

- language - Node.js

- frame work - express
  * test frame work - jest
  * ORM - seaquelize

- SessionDB - Redis

## フロントエンド
- main frame work - Next.js
- css frame work - tailwindcss
- ui frame work - material ui
